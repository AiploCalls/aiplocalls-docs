---
title: "Получаване на изходящи асистенти"
api: "GET /user/assistants/outbound"
description: "Извличане на всички изходящи асистенти за удостоверения потребител"
---

Този endpoint връща списък с всички изходящи асистенти, принадлежащи на удостоверения потребител. Изходящите асистенти се използват за извършване на повиквания до потенциални клиенти и проспекти.

### Response полета

<ResponseField name="data" type="array">
  <Expandable title="properties">
    <ResponseField name="id" type="integer">
      Уникалният идентификатор на асистента
    </ResponseField>

    <ResponseField name="user_id" type="integer">
      ID на потребителя, който притежава този асистент
    </ResponseField>

    <ResponseField name="name" type="string">
      Името на асистента
    </ResponseField>

    <ResponseField name="type" type="string">
      Типът на асистента (винаги ще бъде "outbound" за този endpoint)
    </ResponseField>

    <ResponseField name="mode" type="string">
      Режимът на engine (`pipeline` или `multimodal`)
    </ResponseField>

    <ResponseField name="status" type="string">
      Текущият статус на асистента
    </ResponseField>

    <ResponseField name="voice_id" type="integer">
      ID на гласа, използван от асистента
    </ResponseField>

    <ResponseField name="language_id" type="integer">
      ID на езика, използван от асистента
    </ResponseField>

    <ResponseField name="timezone" type="string">
      Настройката за часова зона на асистента
    </ResponseField>

    <ResponseField name="initial_message" type="string">
      Първоначалното съобщение, което асистентът ще произнесе
    </ResponseField>

    <ResponseField name="system_prompt" type="string">
      Системният prompt, който дефинира поведението на асистента
    </ResponseField>

    <ResponseField name="variables" type="object">
      Персонализирани променливи, дефинирани за асистента
    </ResponseField>

    <ResponseField name="webhook_url" type="string">
      URL адресът на webhook за известия след разговор
    </ResponseField>

    <ResponseField name="is_webhook_active" type="boolean">
      Дали webhook известията са активирани
    </ResponseField>

    <ResponseField name="created_at" type="string">
      Датата и часът, когато асистентът е създаден
    </ResponseField>

    <ResponseField name="updated_at" type="string">
      Датата и часът, когато асистентът е последно актуализиран
    </ResponseField>
  </Expandable>
</ResponseField>

### Бележки

- Този endpoint връща само асистенти с тип "outbound"
- Връщат се всички асистенти, принадлежащи на удостоверения потребител (без пагинация)  
- Използвайте тези асистенти с endpoint за извършване на повиквания, за да инициирате изходящи разговори
- Изходящите асистенти могат да се използват за търговски разговори, проследявания, проучвания и други проактивни сценарии за обаждане
