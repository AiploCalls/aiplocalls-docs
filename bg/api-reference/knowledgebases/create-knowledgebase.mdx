---
title: "Създаване на база знания"
api: "POST /user/knowledgebases"
description: "Създаване на нова база знания"
---

Този endpoint създава нова база знания. След създаването можете да добавите документи към нея, използвайки [създаване на документ](/api-reference/knowledgebases/create-document) endpoint.

### Request Body

<ParamField body="name" type="string" required>
  Името на базата знания (максимум 255 символа)
</ParamField>

<ParamField body="description" type="string">
  Незадължително описание на базата знания (максимум 255 символа)
</ParamField>

### Response

<ResponseField name="message" type="string">
  Съобщение за успех
</ResponseField>

<ResponseField name="data" type="object">
  Създаденият обект база знания
  
  <Expandable title="data свойства">
    <ResponseField name="id" type="integer">
      Уникалният идентификатор на базата знания
    </ResponseField>
    
    <ResponseField name="name" type="string">
      Името на базата знания
    </ResponseField>
    
    <ResponseField name="description" type="string">
      Описанието на базата знания
    </ResponseField>
    
    <ResponseField name="status" type="string">
      Текущо състояние (ще бъде `empty` за нови бази знания)
    </ResponseField>
    
    <ResponseField name="status_label" type="string">
      Четим етикет за състоянието
    </ResponseField>
    
    <ResponseField name="created_at" type="string">
      ISO 8601 времева марка на създаване
    </ResponseField>
    
    <ResponseField name="updated_at" type="string">
      ISO 8601 времева марка на последното актуализиране
    </ResponseField>
  </Expandable>
</ResponseField>

### Следващи стъпки

След създаване на база знания, ще искате да:

1. **Добавите документи** - Използвайте [създаване на документ](/api-reference/knowledgebases/create-document) endpoint за добавяне на съдържание
2. **Изчакате обработката** - Документите се обработват асинхронно
3. **Прикачите към асистент** - Използвайте [актуализиране на асистент](/api-reference/assistants/update-assistant) endpoint за прикачване на базата знания
