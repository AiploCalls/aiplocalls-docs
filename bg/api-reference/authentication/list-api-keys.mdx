---
title: "Списък на API ключовете"
api: "GET /user/api-keys"
description: "Получаване на всички API ключове за автентикирания потребител"
---

Този endpoint връща списък с всички API ключове, свързани с акаунта на автентикирания потребител.

### Отговор

<ResponseField name="api_keys" type="array">
  Масив от API ключ обекти
  
  <Expandable title="api_key свойства">
    <ResponseField name="id" type="integer">
      Уникалният идентификатор на API ключа
    </ResponseField>
    
    <ResponseField name="name" type="string">
      Името/етикетът на API ключа
    </ResponseField>
    
    <ResponseField name="last_used_at" type="string">
      ISO 8601 времева отметка на последното използване на API ключа (null ако никога не е използван)
    </ResponseField>
    
    <ResponseField name="created_at" type="string">
      ISO 8601 времева отметка на създаването на API ключа
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseExample>

```json 200 Response
{
  "api_keys": [
    {
      "id": 1,
      "name": "Production API Key",
      "last_used_at": "2025-01-08T15:30:00.000000Z",
      "created_at": "2025-01-01T10:00:00.000000Z"
    },
    {
      "id": 2,
      "name": "Development API Key",
      "last_used_at": null,
      "created_at": "2025-01-05T14:20:00.000000Z"
    }
  ]
}
```

```json 200 Empty Response
{
  "api_keys": []
}
```

</ResponseExample>

<Note>
От съображения за сигурност действителните API ключ токени не се връщат. Показват се само метаданните за ключовете.
</Note>
