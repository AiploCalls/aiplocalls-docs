---
title: "Освобождаване на телефонен номер"
api: "DELETE /user/phone-numbers/{id}"
description: "Освобождаване на телефонен номер и отказ на абонамента му"
---

Този endpoint освобождава телефонен номер от вашия акаунт. За dedicated номера с абонаменти, това отменя абонамента и номерът ще бъде освободен в края на текущия период на таксуване.

### Path Parameters

<ParamField path="id" type="integer" required>
  Уникалният идентификатор на телефонния номер за освобождаване
</ParamField>

### Response

<ResponseField name="message" type="string">
  Съобщение за успех, указващо статуса на освобождаването
</ResponseField>

<ResponseExample>

```json 200 Subscription Cancelled
{
  "message": "Phone number released successfully. It will be removed at the end of the billing period."
}
```

```json 200 Immediately Deleted
{
  "message": "Phone number deleted successfully."
}
```

```json 400 Already Scheduled
{
  "error": "Phone number is already scheduled for release."
}
```

```json 404 Not Found
{
  "error": "Phone number not found."
}
```

```json 500 Error
{
  "error": "Failed to release phone number. Please try again or contact support."
}
```

</ResponseExample>

### Поведение при освобождаване

Поведението при освобождаване зависи от типа на телефонния номер:

| Тип | Поведение |
|------|----------|
| **Dedicated (нормален)** | Абонаментът се отменя. Номерът остава активен до края на периода на таксуване, след което се освобождава. |
| **SIP** | Веднага се изтрива от вашия акаунт. |
| **Caller ID** | Веднага се изтрива от вашия акаунт и се премахва от телефонния доставчик. |

<Warning>
След като телефонен номер бъде освободен, може да не можете да получите същия номер обратно. Уверете се, че актуализирате всички асистенти или кампании, използващи този номер, преди да го освободите.
</Warning>

### Какво се случва с прикачените ресурси

Когато освободите телефонен номер:
- Всички **inbound асистенти**, прикачени към този номер, ще имат неприсвоен телефонен номер
- Всички **кампании**, използващи този номер, ще го имат премахнат от тяхната ротация на телефонни номера
- **Историята на обаждания** се запазва и остава достъпна
