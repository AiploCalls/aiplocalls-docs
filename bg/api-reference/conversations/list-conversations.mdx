---

Този endpoint връща пагиниран списък на разговорите, принадлежащи на асистентите на автентифицирания потребител. Използвайте го за показване на история на разговорите, филтриране по тип или интеграция с вашия CRM.

<Note>
  Този endpoint изисква автентификация. Подайте вашия API ключ в `Authorization` заглавката като Bearer token.
</Note>

### Query параметри

<ParamField query="type" type="string" optional>
  Филтриране на разговори по тип. Възможни стойности: `test`, `widget`, `whatsapp`, `api`
</ParamField>

<ParamField query="assistant_id" type="integer" optional>
  Филтриране на разговори по ID на асистент (трябва да принадлежи на автентифицирания потребител)
</ParamField>

<ParamField query="date_from" type="string" optional>
  Филтриране на разговори от тази дата (YYYY-MM-DD формат)
</ParamField>

<ParamField query="date_to" type="string" optional>
  Филтриране на разговори до тази дата (YYYY-MM-DD формат)
</ParamField>

<ParamField query="per_page" type="integer" optional>
  Брой разговори на страница (1-100, по подразбиране: 15)
</ParamField>

<ParamField query="page" type="integer" optional>
  Номер на страницата (по подразбиране: 1)
</ParamField>

### Полета в отговора

<ResponseField name="data" type="array">
  <Expandable title="properties">
    <ResponseField name="id" type="string">
      Уникалният UUID идентификатор на разговора
    </ResponseField>

    <ResponseField name="assistant_id" type="string">
      UUID на асистента, който обработва този разговор
    </ResponseField>

    <ResponseField name="assistant_name" type="string">
      Името на асистента, който обработва този разговор
    </ResponseField>

    <ResponseField name="type" type="string">
      Типът на разговора: `test`, `widget`, `whatsapp`, или `api`
    </ResponseField>

    <ResponseField name="message_count" type="integer">
      Общ брой съобщения в разговора
    </ResponseField>

    <ResponseField name="total_cost" type="number">
      Общата цена на разговора в USD
    </ResponseField>

    <ResponseField name="ai_enabled" type="boolean">
      Дали AI отговорите са разрешени за този разговор
    </ResponseField>

    <ResponseField name="created_at" type="string">
      Датата и часът, когато разговорът е създаден
    </ResponseField>

    <ResponseField name="updated_at" type="string">
      Датата и часът, когато разговорът е последно обновен
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="current_page" type="integer">
  Номерът на текущата страница
</ResponseField>

<ResponseField name="per_page" type="integer">
  Брой елементи на страница
</ResponseField>

<ResponseField name="total" type="integer">
  Общ брой разговори, отговарящи на критериите
</ResponseField>

<ResponseField name="last_page" type="integer">
  Номерът на последната страница
</ResponseField>

```python Python
import requests

response = requests.get(
    'https://call.aiployees.com/api/user/conversations',

    params={
        'type': 'widget',
        'per_page': 10

print(data['data'])
```

## Типове разговори

| Тип | Описание |
|------|-------------|
| `test` | Вътрешни тестови разговори от интерфейса за тестване на асистента |
| `widget` | Разговори от web chat widget-а |
| `whatsapp` | WhatsApp Business разговори |
| `api` | Разговори, създадени чрез API |

## Случаи на употреба

- **Analytics табло**: Показване на метрики и тенденции на разговорите
- **CRM интеграция**: Синхронизация на данните за разговорите с вашата клиентска база данни
- **Мониторинг на качеството**: Преглед на обемите разговори по тип и асистент
- **Преглед на фактурирането**: Проследяване на разходите за разговори в цялата ви организация
