---
title: "Актуализиране на документ"
api: "PUT /user/knowledgebases/{knowledgebaseId}/documents/{documentId}"
description: "Актуализиране на името или описанието на документ"
---

Този endpoint актуализира метаданните на съществуващ документ. Можете да актуализирате само името и описанието - типът и съдържанието на документа не могат да бъдат променени.

### Path Параметри

<ParamField path="knowledgebaseId" type="integer" required>
  Уникалният идентификатор на базата знания
</ParamField>

<ParamField path="documentId" type="integer" required>
  Уникалният идентификатор на документа за актуализиране
</ParamField>

### Request Body

<ParamField body="name" type="string">
  Новото име на документа (максимум 255 символа)
</ParamField>

<ParamField body="description" type="string">
  Новото описание на документа (максимум 255 символа)
</ParamField>

### Response

<ResponseField name="message" type="string">
  Съобщение за успех
</ResponseField>

<ResponseField name="data" type="object">
  Актуализираният обект документ
  
  <Expandable title="data свойства">
    <ResponseField name="id" type="integer">
      Уникалният идентификатор на документа
    </ResponseField>
    
    <ResponseField name="name" type="string">
      Името на документа
    </ResponseField>
    
    <ResponseField name="description" type="string">
      Описание на документа
    </ResponseField>
    
    <ResponseField name="type" type="string">
      Тип на документа
    </ResponseField>
    
    <ResponseField name="type_label" type="string">
      Четимо за хора наименование на типа
    </ResponseField>
    
    <ResponseField name="status" type="string">
      Статус на обработката
    </ResponseField>
    
    <ResponseField name="status_label" type="string">
      Четимо за хора наименование на статуса
    </ResponseField>
    
    <ResponseField name="updated_at" type="string">
      ISO 8601 timestamp на последната актуализация
    </ResponseField>
  </Expandable>
</ResponseField>

<Note>
За да актуализирате съдържанието на документ, трябва да изтриете съществуващия документ и да създадете нов с актуализираното съдържание.
</Note>
