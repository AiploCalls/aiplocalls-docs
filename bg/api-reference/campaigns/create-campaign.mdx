---
title: "Създаване на кампания"
api: "POST /user/campaign"
description: "Създаване на нова кампания за изходящи обаждания"
---

Този endpoint ви позволява да създадете нова кампания за изходящи обаждания с указаната конфигурация.

### Request body

<ParamField body="name" type="string" required>
  Името на кампанията. Максимум 255 символа.
</ParamField>

<ParamField body="assistant_id" type="integer" required>
  ID на асистента, който да се използва за кампанията. Трябва да бъде асистент, способен за изходящи обаждания.
</ParamField>

<ParamField body="timezone" type="string">
  Идентификатор на часовата зона за кампанията (напр. `America/New_York`, `Europe/London`). По подразбиране е часовата зона на вашия акаунт.
</ParamField>

<ParamField body="max_calls_in_parallel" type="integer" default="3">
  Максимален брой едновременни обаждания. Минимум: 1. Максимумът зависи от лимита за паралелни обаждания на вашия план (до 10).
</ParamField>

<ParamField body="allowed_hours_start_time" type="string" default="00:00">
  Начало на разрешения прозорец за обаждания в формат `H:i` (напр. `09:00`).
</ParamField>

<ParamField body="allowed_hours_end_time" type="string" default="23:59">
  Край на разрешения прозорец за обаждания в формат `H:i` (напр. `17:00`).
</ParamField>

<ParamField body="allowed_days" type="array" default="всичките 7 дни">
  Масив от имена на дни от седмицата, когато са разрешени обаждания. Валидни стойности: `monday`, `tuesday`, `wednesday`, `thursday`, `friday`, `saturday`, `sunday`.
</ParamField>

<ParamField body="max_retries" type="integer" default="3">
  Максимален брой опити за повторение при неуспешни обаждания. Диапазон: 1-5.
</ParamField>

<ParamField body="retry_interval" type="integer" default="60">
  Интервал в минути между опитите за повторение. Диапазон: 10-4320 (до 3 дни).
</ParamField>

<ParamField body="retry_on_voicemail" type="boolean">
  Дали да се повтарят обаждания, които са достигнали гласова поща.
</ParamField>

<ParamField body="retry_on_goal_incomplete" type="boolean">
  Дали да се повтарят обаждания, при които целта не е постигната.
</ParamField>

<ParamField body="goal_completion_variable" type="string">
  Име на boolean променлива от post-call схемата на вашия асистент за проследяване постигането на целта. Максимум 255 символа.
</ParamField>

<ParamField body="mark_complete_when_no_leads" type="boolean" default="true">
  Дали автоматично да се маркира кампанията като завършена, когато няма повече оставащи потенциални клиенти за обаждане.
</ParamField>

<ParamField body="phone_number_ids" type="array">
  Масив от ID-та на телефонни номера за използване в кампанията. Всяко ID трябва да бъде отделно цяло число.
</ParamField>

### Response

<ResponseField name="message" type="string">
  Съобщение за успех, потвърждаващо, че кампанията е създадена
</ResponseField>

<ResponseField name="data" type="object">
  Данните на създадената кампания
  <Expandable title="data properties">
    <ResponseField name="id" type="integer">
      ID на създадената кампания
    </ResponseField>

    <ResponseField name="name" type="string">
      Името на кампанията
    </ResponseField>

    <ResponseField name="status" type="string">
      Статусът на кампанията (започва като `draft`)
    </ResponseField>

    <ResponseField name="max_calls_in_parallel" type="integer">
      Максимален брой едновременни обаждания
    </ResponseField>

    <ResponseField name="mark_complete_when_no_leads" type="boolean">
      Дали да се маркира кампанията като завършена, когато няма потенциални клиенти
    </ResponseField>

    <ResponseField name="allowed_hours_start_time" type="string">
      Начало на разрешения прозорец за обаждания
    </ResponseField>

    <ResponseField name="allowed_hours_end_time" type="string">
      Край на разрешения прозорец за обаждания
    </ResponseField>

    <ResponseField name="allowed_days" type="array">
      Дни от седмицата, когато са разрешени обаждания
    </ResponseField>

    <ResponseField name="max_retries" type="integer">
      Максимален брой опити за повторение
    </ResponseField>

    <ResponseField name="retry_interval" type="integer">
      Интервал в минути между повторенията
    </ResponseField>

    <ResponseField name="created_at" type="string">
      Времева отметка, когато кампанията е създадена
    </ResponseField>

    <ResponseField name="updated_at" type="string">
      Времева отметка, когато кампанията е последно актуализирана
    </ResponseField>
  </Expandable>
</ResponseField>

### Error Responses

<ResponseField name="403 Forbidden">
  <Expandable title="Error Response">
    <ResponseField name="message" type="string">
      Съобщение за грешка, когато потребителят е достигнал лимита на кампании за своя план
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="404 Not Found">
  <Expandable title="Error Response">
    <ResponseField name="message" type="string">
      Съобщение за грешка, когато указаният асистент не е намерен или не е асистент за изходящи обаждания
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="422 Validation Error">
  <Expandable title="Error Response">
    <ResponseField name="message" type="string">
      Съобщение за грешка, указващо неуспешна валидация
    </ResponseField>
    <ResponseField name="errors" type="object">
      Подробни грешки от валидацията за всяко поле
    </ResponseField>
  </Expandable>
</ResponseField>
