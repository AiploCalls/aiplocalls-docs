---

Този endpoint връща списък с всички налични гласове, които могат да бъдат използвани при създаване или обновяване на асистенти, с възможност за филтриране по режим на асистента и език.

### Query Parameters

<ParamField query="mode" type="string">
  Филтриране на гласове по режим на асистента. Опции: `pipeline`, `multimodal`, `dualplex`
</ParamField>

<ParamField query="language_id" type="integer">
  Филтриране на гласове по ID на езика. Използвайте endpoint-а [Get Languages](/api-reference/assistants/get-languages) за да получите наличните ID на езици.
</ParamField>

### Response fields

<ResponseField name="data" type="array">
  <Expandable title="properties">
    <ResponseField name="id" type="integer">
      Уникалният идентификатор на гласа
    </ResponseField>

    <ResponseField name="name" type="string">
      Показваното име на гласа
    </ResponseField>

    <ResponseField name="language_id" type="integer">
      ID на езика, с който е свързан този глас
    </ResponseField>

    <ResponseField name="gender" type="string">
      Полът на гласа (male/female)
    </ResponseField>

    <ResponseField name="preview_url" type="string">
      URL към аудио образец на гласа. Може да бъде използван за предварителното прослушване на гласове преди избиране на един.
    </ResponseField>

    <ResponseField name="accent" type="string">
      Акцентът на гласа (например "American", "British"). Може да бъде `null`.
    </ResponseField>

    <ResponseField name="age" type="string">
      Възрастовата категория на гласа (например "Young", "Middle aged"). Може да бъде `null`.
    </ResponseField>

    <ResponseField name="synthesizer_provider_id" type="integer">
      ID на TTS провайдъра, към който принадлежи този глас. Използвайте endpoint-а [Get Synthesizer Providers](/api-reference/assistants/get-synthesizer-providers) за да разрешите имената на провайдърите. Може да бъде `null`.
    </ResponseField>
  </Expandable>
</ResponseField>

### Забележки

- Използвайте полето `id` като `voice_id` при създаване или обновяване на асистенти
- Филтрирайте по `mode` за да получите само гласове, съвместими с планирания от вас тип асистент
- Филтрирайте по `language_id` за да получите гласове за конкретен език
- Ако не са предоставени филтри, се връщат всички налични гласове
- Връщат се само гласове, които са публични или принадлежат на удостоверения потребител
