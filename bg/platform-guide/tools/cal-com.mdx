## Настройка на Cal.com

1. Отидете на уебсайта [Cal.com](https://cal.com/) и се регистрирайте за акаунт
2. Създайте нов тип събитие, например "Среща с AI асистент"
3. Копирайте ID на събитието като кликнете върху типа събитие и копирате този номер от URL адреса

<img height="200" src="/resources/cal-event-id.png" />

4. Създайте API ключ в Cal.com

## Добавете инструмента към вашия асистент

Добавете този инструмент към вашия асистент и предайте ID на събитието и API ключа.

<img height="200" src="/resources/cal-tool.png" />
