Този endpoint ви позволява да направите телефонно обаждане.

### Request body

<ParamField body="phone_number" type="string" required>
  Телефонният номер на клиента в E.164 формат (напр. +1234567890)
</ParamField>

<ParamField body="assistant_id" type="integer" required>
  ID-то на асистента, с който да се направи обаждането
</ParamField>

<ParamField body="variables" type="object">
  Персонализирани променливи за предаване към обаждането. Можете да включите **всички променливи, които са дефинирани във вашия асистент** (вижте [Променливи на обаждането](/ai-assistants/settings/prompt-and-tools#call-variables)). Примерите по-долу са само за илюстрация — използвайте вашите собствени имена на променливи, както са конфигурирани в настройките на асистента.

  <Expandable title="Примерни променливи (добавете свои собствени)">
    <ParamField body="customer_name" type="string">
      напр. името на клиента
    </ParamField>

    <ParamField body="email" type="string">
      напр. имейлът на клиента
    </ParamField>
  </Expandable>
</ParamField>

### Response

<ResponseField name="message" type="string" default="Call initiated successfully">
  Съобщението от отговора
</ResponseField>

```json 200 Response
{
  "message": "Call initiated successfully",
  "data": {}
}
```

</ResponseExample>
