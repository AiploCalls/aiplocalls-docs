description: "Изтриване на база знания и всичките й документи"
---

Този endpoint окончателно изтрива база знания и всичките й свързани документи. Това действие не може да бъде отменено.

### Параметри в пътя

<ParamField path="id" type="integer" required>
  Уникалният идентификатор на базата знания за изтриване
</ParamField>

### Отговор

<ResponseField name="message" type="string">
  Съобщение за успех
</ResponseField>

<ResponseExample>

```json 200 Response
{
  "message": "Knowledgebase deleted successfully."
}
```

```json 404 Not Found
{
  "error": "Knowledgebase not found."
}
```

</ResponseExample>

### Какво се изтрива

Когато изтриете база знания:

- **Всички документи** в базата знания се изтриват окончателно
- **Vector embeddings** се премахват от векторната база данни
- **Данните от хранилището на документи** се почистват от външните услуги

### Какво се случва със свързаните асистенти

Всички асистенти, които са използвали тази база знания, ще имат премахната референцията към базата знания. Те ще продължат да функционират, но без контекста на базата знания.

<Warning>
Това действие е необратимо. Всички документи и тяхното обработено съдържание ще бъдат изтрити окончателно. Уверете се, че искате да изтриете тази база знания, преди да продължите.
</Warning>
