---
title: "Актуализиране на база знания"
api: "PUT /user/knowledgebases/{id}"
description: "Актуализиране на съществуваща база знания"
---

Този endpoint актуализира името и/или описанието на съществуваща база знания.

### Path Parameters

<ParamField path="id" type="integer" required>
  Уникалният идентификатор на базата знания за актуализиране
</ParamField>

### Request Body

<ParamField body="name" type="string">
  Новото име за базата знания (максимум 255 символа)
</ParamField>

<ParamField body="description" type="string">
  Новото описание за базата знания (максимум 255 символа)
</ParamField>

### Response

<ResponseField name="message" type="string">
  Съобщение за успех
</ResponseField>

<ResponseField name="data" type="object">
  Актуализираният обект база знания
  
  <Expandable title="data properties">
    <ResponseField name="id" type="integer">
      Уникалният идентификатор на базата знания
    </ResponseField>
    
    <ResponseField name="name" type="string">
      Името на базата знания
    </ResponseField>
    
    <ResponseField name="description" type="string">
      Описание на базата знания
    </ResponseField>
    
    <ResponseField name="status" type="string">
      Текущ статус
    </ResponseField>
    
    <ResponseField name="status_label" type="string">
      Четимо за човека наименование на статуса
    </ResponseField>
    
    <ResponseField name="updated_at" type="string">
      ISO 8601 времева отметка на последната актуализация
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseExample>

```json 200 Response
{
  "message": "Knowledgebase updated successfully.",
  "data": {
    "id": 1,
    "name": "Updated Product Documentation",
    "description": "Updated description for our technical docs",
    "status": "active",
    "status_label": "Active",
    "updated_at": "2025-01-08T15:45:00.000000Z"
  }
}
```

```json 404 Not Found
{
  "error": "Knowledgebase not found."
}
```

```json 422 Validation Error
{
  "message": "The name may not be greater than 255 characters.",
  "errors": {
    "name": [
      "The name may not be greater than 255 characters."
    ]
  }
}
```

</ResponseExample>

<Note>
Актуализирането на името или описанието на база знания не засяга нейните документи или статуса на обработката.
</Note>
