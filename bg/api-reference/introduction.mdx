Добре дошли в документацията на Aiplocalls API. Нашият API ви позволява програмно да взаимодействате с платформата Aiplocalls, давайки ви възможност да създавате и управлявате потенциални клиенти, да планирате обаждания и много други.

## Base URL

Всички API заявки трябва да се правят към следния base URL:

```
https://call.aiployees.com/api
```
