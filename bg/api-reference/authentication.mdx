Aiplocalls API използва API ключове за автентификация на заявките. Трябва да включите вашия API ключ в заглавката на всяка заявка, която правите към API.

## Получаване на API ключ

За да получите API ключ:

1. Влезте в вашия Aiplocalls акаунт на [https://call.aiployees.com](https://call.aiployees.com)
2. Отидете на страницата "API Keys"
3. Кликнете на бутона "Create new API Key"
4. Копирайте вашия нов API ключ и го съхранете сигурно

<Warning>
  Пазете вашия API ключ поверителен и сигурен. Не го споделяйте публично или го
  качвайте в системи за контрол на версиите.
</Warning>

## Използване на API ключа

Включете вашия API ключ в заглавката `Authorization` на всяка заявка:

```
Authorization: Bearer YOUR_API_KEY
```

Заменете `YOUR_API_KEY` с вашия действителен API ключ.

## Примерна заявка

Ето пример за това как да включите API ключа в cURL заявка:

```bash
curl -X GET "https://call.aiployees.com/api/user/campaigns" \
     -H "Authorization: Bearer YOUR_API_KEY"
```

Заменете `YOUR_API_KEY` с вашия действителен API ключ при правенето на заявки.
