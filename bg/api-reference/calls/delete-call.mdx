---
title: "Изтриване на обаждане"
api: "DELETE /user/calls/{call}"
description: "Изтриване на конкретен запис за обаждане"
---

Този endpoint ви позволява да изтриете конкретен запис за обаждане, който принадлежи на удостоверения потребител.

### Path Parameters

<ParamField path="call" type="integer" required>
  Уникалният идентификатор на обаждането за изтриване
</ParamField>

### Response

<ResponseField name="message" type="string">
  Потвърдително съобщение, указващо че обаждането е изтрито успешно
</ResponseField>

### Error Responses

<ResponseField name="404 Not Found">
  <Expandable title="Отговор при грешка">
    <ResponseField name="message" type="string">
      Съобщение за грешка, когато обаждането не е намерено или не принадлежи на удостоверения потребител
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseExample>

```json 200 Success Response
{
  "message": "Call deleted successfully"
}
```

```json 404 Not Found
{
  "message": "Call not found"
}
```

</ResponseExample>

### Забележки

- Могат да се изтриват само обаждания, които принадлежат на асистенти, собственост на удостоверения потребител
- След като обаждането е изтрито, то не може да бъде възстановено
- Това действие ще премахне окончателно записа за обаждането, включително всякакви свързани препис, запис и метаданни
- Файлът със записа на обаждането (ако съществува) също ще бъде изтрит от хранилището
