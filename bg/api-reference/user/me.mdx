title: "Получаване на текущия потребител"

description: "Получаване на профилната информация на автентикирания потребител"
---

Този endpoint връща профилната информация за текущо автентикирания потребител, включително баланса на неговия акаунт.

### Response

<ResponseField name="name" type="string">
  Пълното име на потребителя
</ResponseField>

<ResponseField name="email" type="string">
  Имейл адресът на потребителя
</ResponseField>

<ResponseField name="total_balance" type="number">
  Текущият баланс на акаунта на потребителя в USD
</ResponseField>

<ResponseExample>

```json 200 Response
{
  "name": "John Doe",
  "email": "john.doe@example.com",
  "total_balance": 125.50
}
```

</ResponseExample>

### Случаи на използване

Този endpoint е полезен за:

- **Проверка на автентикацията** - Потвърждаване, че API ключът е валиден и принадлежи на очаквания потребител
- **Проверка на баланса на акаунта** - Наблюдение на наличните кредити преди извършване на API повиквания
- **Показване на потребителска информация** - Показване на текущия потребител в UI на вашето приложение
