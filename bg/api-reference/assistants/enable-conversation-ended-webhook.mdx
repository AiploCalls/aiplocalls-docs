---
title: "Активиране на webhook за приключена беседа"
api: "POST /user/assistants/enable-conversation-ended-webhook"
description: "Активиране на webhook уведомления когато чат беседа приключи"
---

Този endpoint активира webhook уведомления за приключена беседа за асистент, позволявайки ви да получавате актуализации в реално време когато чат беседа (WhatsApp или Web Widget) приключи. Webhook-ът включва пълната транскрипция, извлечените променливи и информация за клиента.

### Тяло на заявката

<ParamField body="assistant_id" type="integer" required>
  ID-то на асистента за който да се активира webhook-ът за приключена беседа
</ParamField>

<ParamField body="webhook_url" type="string" required>
  URL-то където ще се изпращат webhook уведомленията когато беседа приключи
</ParamField>

### Отговор

<ResponseField name="message" type="string">
  Съобщение за успех потвърждаващо че webhook-ът е активиран
</ResponseField>

<ResponseField name="data" type="array">
  Празен масив (запазен за бъдеща употреба)
</ResponseField>

### Грешни отговори

<ResponseField name="404 Not Found">
  <Expandable title="Отговор за грешка">
    <ResponseField name="message" type="string">
      Съобщение за грешка когато асистентът не е намерен или не принадлежи на автентикирания потребител
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="422 Validation Error">
  <Expandable title="Отговор за грешка">
    <ResponseField name="message" type="string">
      Съобщение за грешка указващо неуспешна валидация
    </ResponseField>
    <ResponseField name="errors" type="object">
      Детайлни грешки от валидацията за всяко поле
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseExample>

```json 200 Success Response
{
  "message": "Conversation ended webhook enabled successfully",
  "data": []
}
```

```json 404 Not Found
{
  "message": "Assistant not found"
}
```

```json 422 Validation Error
{
  "message": "The given data was invalid.",
  "errors": {
    "assistant_id": [
      "The assistant id field is required."
    ],
    "webhook_url": [
      "The webhook url field is required."
    ]
  }
}
```

</ResponseExample>
