---
title: "Получаване на разговор"
api: "GET /conversations/{uuid}"
description: "Извличане на историята на съобщенията от разговор"
---

Този endpoint извлича пълната история на съобщенията от съществуващ разговор. Използвайте го за показване на предишни съобщения при възобновяване на разговор или за преглед на съдържанието на разговора.

### Path параметри

<ParamField path="uuid" type="string" required>
  Уникалният UUID идентификатор на разговора за извличане
</ParamField>

### Полета в отговора

<ResponseField name="status" type="boolean">
  Указва дали заявката е била успешна
</ResponseField>

<ResponseField name="history" type="array">
  Масив от съобщения в разговора, подредени хронологично (най-старите първи).

  <Expandable title="Свойства на обекта съобщение">
    <ResponseField name="role" type="string">
      Ролята на съобщението: `assistant` или `user`
    </ResponseField>
    <ResponseField name="content" type="string">
      Съдържанието на съобщението
    </ResponseField>
    <ResponseField name="function_calls" type="array">
      Незадължително. Масив от function calls направени от асистента по време на това съобщение (напр. резервации в календар, API повиквания).
    </ResponseField>
  </Expandable>
</ResponseField>

### Отговори при грешка

<ResponseField name="status" type="boolean">
  Ще бъде `false` когато възникне грешка
</ResponseField>

<ResponseField name="error" type="string">
  Съобщение за грешка. Възможни стойности:
  - `Conversation not found` - Предоставеният UUID не съответства на никой разговор
</ResponseField>

<RequestExample>

```bash cURL
curl -X GET "https://call.aiployees.com/api/conversations/7c9e6679-7425-40de-944b-e07fc1f90ae7"
```

```javascript JavaScript
const conversationId = '7c9e6679-7425-40de-944b-e07fc1f90ae7';

const response = await fetch(
  `https://call.aiployees.com/api/conversations/${conversationId}`
);

const data = await response.json();
console.log(data.history);
```

```python Python
import requests

conversation_id = '7c9e6679-7425-40de-944b-e07fc1f90ae7'

response = requests.get(
    f'https://call.aiployees.com/api/conversations/{conversation_id}'
)

data = response.json()
print(data['history'])
```

## Случаи на употреба

- **Възобновяване на разговори**: Извличане на история когато потребител се връща за да продължи разговор
- **Аналитика**: Преглед на съдържанието на разговора за осигуряване на качество
- **Интеграция**: Синхронизиране на данни от разговор с вашата CRM или support система
