---
title: "Изтриване на инструмент по време на разговор"
api: "DELETE /user/tools/{id}"
description: "Изтриване на инструмент по време на разговор"
---

Този endpoint ви позволява окончателно да изтриете инструмент по време на разговор. Това също така ще премахне инструмента от всички асистенти, на които е бил присвоен.

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token за автентикация
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Трябва да бъде `application/json`
</ParamField>

<ParamField header="Accept" type="string" required>
  Трябва да бъде `application/json`
</ParamField>

### Path Parameters

<ParamField path="id" type="integer" required>
  Уникалният идентификатор на инструмента за изтриване
</ParamField>

### Response fields

<ResponseField name="message" type="string">
  Съобщение за успех, потвърждаващо изтриването
</ResponseField>

<ResponseExample>

```json 200 OK
{
  "message": "Tool deleted successfully"
}
```

```json 404 Not Found
{
  "message": "Tool not found"
}
```

</ResponseExample>

### Забележка

Когато инструмент се изтрие, той автоматично се премахва от всички асистенти. За да управлявате присвояването на инструменти без да изтривате инструмента, използвайте:

- **[Update Assistant](/api-reference/assistants/update-assistant)** - Използвайте параметъра `tool_ids`, за да премахнете инструменти от конкретни асистенти
