---
---
title: "Актуализиране на лид"
api: "PUT /user/leads/{id}"
description: "Актуализирайте съществуващ лид във вашите кампании"
---

Актуализирайте лид, като предоставите ID на лида и полетата, които искате да промените. Само полетата, предоставени в заявката, ще бъдат актуализирани.

## Path Parameters

<ParamField path="id" type="integer" required>
  ID на лида за актуализиране
</ParamField>

## Body Parameters

<ParamField body="campaign_id" type="integer" optional>
  ID на кампанията, към която да бъде присвоен лидът
</ParamField>

<ParamField body="phone_number" type="string" optional>
  Телефонният номер на лида (ще бъде форматиран до E164)
</ParamField>

<ParamField body="status" type="string" optional>
  Статусът на лида. Трябва да бъде един от: `created`, `completed`, `reached-max-retries`
</ParamField>

<ParamField body="variables" type="object" optional>
  Персонализирани променливи за сливане със съществуващите променливи на лида
</ParamField>

## Response

<ResponseField name="message" type="string">
  Съобщение за успех, показващо че лидът е актуализиран
</ResponseField>

<RequestExample>

```bash cURL
curl -X PUT "https://call.aiployees.com/api/user/leads/123" \
  -H "Authorization: Bearer YOUR_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "campaign_id": 456,
    "phone_number": "+1234567890",
    "variables": {
      "name": "John Doe",
      "company": "Acme Corp"
    }
  }'
```

</RequestExample>

<ResponseExample>

```json Success Response
{
  "message": "Lead updated successfully"
}
```

```json Error Response - Lead Not Found
{
  "message": "Lead not found"
}
```

```json Error Response - Campaign Not Found
{
  "message": "Campaign not found"
}
```

```json Error Response - Invalid Phone Number
{
  "message": "Invalid phone number"
}
```

```json Error Response - Validation Failed
{
  "message": "Validation failed",
  "errors": {
    "campaign_id": ["The selected campaign does not exist"],

    "variables": ["Variables must be an array"]
  }

```

```json Error Response - Invalid Fields
{
  "message": "Invalid fields: invalid_field, another_invalid_field"

```

</ResponseExample>

## Забележки

- Лидът трябва да принадлежи на удостоверения потребител
- Ако актуализирате кампанията, новата кампания също трябва да принадлежи на удостоверения потребител
- Телефонните номера се форматират и валидират автоматично
- Променливите се сливат със съществуващите променливи (не се заменят)
- Само разрешените полета могат да бъдат актуализирани: `campaign_id`, `phone_number`, `status`, `variables`
- Полето `status` има ограничени стойности за целостта на данните
